# 🖥️ Network Dashboard

Dashboard local para visualizar y monitorizar los dispositivos conectados a una red doméstica.

Este proyecto forma parte de mi laboratorio personal [`mikel-labs`](../README.md).

## 🚀 Objetivo

Desarrollar una herramienta full stack que permita:

- Escanear los dispositivos conectados a la red local.
- Monitorizar su disponibilidad en tiempo real.
- Visualizar los datos desde una interfaz web sencilla.

## 📂 Estructura del proyecto

```
network-dashboard/
├── backend/     # Lógica de escaneo, monitorización y API
├── frontend/    # Interfaz web con Vue
├── scripts/     # Scripts de prueba o utilidades
├── docs/        # Documentación del proyecto (roadmap, diseño, etc.)
└── README.md    # Este archivo
```
